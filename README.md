# Парсер статей с Habr.com

Этот скрипт на Python предназначен для парсинга ежедневных топовых статей с сайта Habr.com.

## Функциональность

- Получение списка топовых статей за день
- Извлечение заголовков, количества просмотров и URL статей
- Парсинг полного текста каждой статьи
- Структурированное хранение данных в dataclass

## Установка зависимостей

Перед использованием убедитесь, что установлены все необходимые библиотеки:

pip install -r requirements.txt

## Структура проекта

Основные функции:

get_url_html(url) - получает HTML-код страницы по URL
get_soup(html_text) - создает объект BeautifulSoup для парсинга
get_all_posts(soup) - извлекает данные всех статей со страницы
get_article_text(url) - получает полный текст отдельной статьи

## Скрипт выполнит следующие действия:

Получит HTML главной страницы с топовыми статьями
Распарсит список статей
Для каждой статьи получит полный текст
Выведет структурированные данные в консоль

## Скрипт возвращает список объектов ArticleData с полной информацией о каждой статье, включая:

Название статьи
Количество просмотров
Ссылку на статью
Полный текст статьи
